# FLunchbot
A Slack Bot to make Fancy Lunch decisions

#### Running FLunchbot
* Get a Bot token from Slack:
  -> http://my.slack.com/services/new/bot
* Run your bot from the command line: `token=<MY TOKEN> node flunch_bot.js`
